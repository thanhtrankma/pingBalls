# Balls Ping - React

Dự án React với p5.js mô phỏng va chạm bóng trong vòng tròn.

## Cài đặt

```bash
npm install
```

## Chạy dự án

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tính năng

- Điều chỉnh số lượng bóng (1-30)
- Điều chỉnh tốc độ tối thiểu (2-15)
- Nút Reset để bắt đầu lại
- Hiệu ứng âm thanh khi va chạm
- Hiệu ứng phát sáng khi bóng va chạm với tường
